Lista de definición

Naranja
Fruta citrica con mucha vitamina C
Mango
Fruta con un sabor inigualable, se puede comer biche o maduro, con sal o pimienta molido
Coco
Fruta de tierra seca caracteristicas de playas
Kiwi
Fruta muy utilizada para crear helado
Limón
Fruta acida, muy utilizada para las ensaladas o acompañantes para licor
