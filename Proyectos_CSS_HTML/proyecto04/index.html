Definiciones

  • ordenador> Un ordenador es una máquina que la gente usa para trabajar, estudiar, entretenerse, y comunicarse, a menudo referida como computadora en algunos países de habla hispana.

  • Ordenador Un ordenador es una máquina electrónica programable que procesa datos y realiza operaciones aritméticas y lógicas a gran velocidad. Se compone de hardware (componentes físicos) y software (programas y sistemas operativos).
